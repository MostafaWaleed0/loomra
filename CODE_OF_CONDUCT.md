# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and constructive environment for everyone who interacts with the Loomra project. This Code of Conduct outlines our expectations for participant behavior and the consequences for unacceptable conduct.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub repository (issues, pull requests, code reviews)
- GitHub Discussions
- Any other communication channels related to the project

This Code of Conduct also applies when representing the project in public spaces.

## Expected Behavior

We expect all participants to:

- **Be respectful and considerate** in all interactions
- **Communicate constructively** and provide helpful feedback
- **Focus on what is best** for the project and community
- **Show empathy** towards other community members
- **Accept responsibility** for mistakes and learn from them
- **Be patient** with those who are new or learning
- **Respect different viewpoints** and experiences
- **Follow the project's contribution guidelines** and license terms

## Unacceptable Behavior

The following behaviors are considered unacceptable:

- Harassment, intimidation, or discrimination of any kind
- Offensive, derogatory, or discriminatory comments
- Personal or political attacks
- Trolling, insulting, or sustained disruption
- Publishing others' private information without permission
- Spamming or excessive self-promotion
- Any conduct that could be considered inappropriate in a professional setting
- Violating the project's license terms or attempting to use the code in unauthorized ways

## Contribution Guidelines

Please note that while contributions are welcome:

- All pull requests are subject to review and approval by the maintainer
- By submitting a contribution, you agree to the project's license terms
- The maintainer reserves the right to accept, modify, or reject any contribution
- Contributors may not use the project code for commercial purposes or in competing products (see LICENSE for details)

## Reporting and Enforcement

If you experience or witness unacceptable behavior, or have any concerns, please report it by contacting:

**Email:** mw.dev500@gmail.com

All reports will be handled with discretion and confidentiality.

### Enforcement Process

1. **Review:** Reports will be reviewed promptly and fairly
2. **Investigation:** The maintainer will investigate the situation and gather relevant information
3. **Decision:** Appropriate action will be determined based on the severity and context
4. **Action:** Consequences may include:
   - A warning to the offender
   - Temporary or permanent ban from project spaces
   - Removal of comments, issues, or pull requests
   - Other actions deemed appropriate

### Appeals

If you believe an enforcement decision was made in error, you may appeal by contacting the same email address with additional context.

## Attribution

This Code of Conduct is designed to foster a positive environment for the Loomra project community.
